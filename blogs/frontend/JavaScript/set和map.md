|      | Set                           | Map                                           |
| ---- | ----------------------------- | --------------------------------------------- |
| 概念 | 值的集合，值唯一              | 键值对集合，键唯一                            |
| api  | **add()**,delete(),has(),size | **set(),get()**,delete(),has(),size           |
| 用途 | 去重                          | 数据映射                                      |
| 特性 | 无键，值不能重复              | 键可是任意类型（原始值 / 引用类型），值可重复 |
