### 例一：斐波那契数列

斐波那契数，通常用 F(n) 表示，形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是： F(0) = 0，F(1) = 1 F(n) = F(n - 1) + F(n - 2)，其中 n > 1 给你n ，请计算 F(n) 。

示例 1：

- 输入：2
- 输出：1
- 解释：F(2) = F(1) + F(0) = 1 + 0 = 1

```js
var fib = function (n) {
    //1、确定dp数组，下标含义：dp[i]表示斐波那契数列第i个的数值
    let dp = []
    //2、公式题目已经给出
    //3、dp数组初始化
    dp[0] = 0
    dp[1] = 1
    //遍历顺序从前往后
    for(let i = 2;i <= n;i++) {
        dp[i] = dp[i-1] + dp[i-2]
    }
    return dp[n]
};
```
