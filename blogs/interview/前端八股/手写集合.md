## 手写API（高频）

### 手写call
内置对象：Function (实例方法)

Function.prototype.call()

本质：把函数临时挂载到对象上执行，来改变this指向

```js
// 使用示例
function greet() {
  console.log(this.animal, "的睡眠时间一般在", this.sleepDuration, "之间");
}

const obj = {
  animal: "猫",
  sleepDuration: "12 到 16 小时",
};

greet.call(obj); // 猫 的睡眠时间一般在 12 到 16 小时 之间

// --------------------***❗️手写⚠️***--------------------
Funtion.prototype.myCall = function (context, ...args) {
	if (typeof this !== 'function') {
		throw new TypeError(this + 'is not a function');
	}

	context = context === null ? globalThis : Object(context);
	const fnKey = Symbol();
	context[fnKey] = this;
	const result = context[fnKey](...args);
	delete context[fnKey]; //删掉临时挂上去的函数，避免影响原对象
	return result;
}

```

### 手写bind
和call的区别就是不会立即执行，（所以才会具备柯里化能力，因为可以在执行函数的时候再传一次值）

```js
Function.prototype.myBind = function (context, ...args) {
	if (typeof this !== 'function') {
		throw new TypeError(this + 'is not a function');
	}
	const self = this;
	return function Fn() {
		if(this instanceof Fn) {
			return new self(...args, ...argments);
		}
		return self.apply(context, [...args, ...argments]);
	}
}
```

### 手写instanceof
type：运算符

```js
function myInstanceof(obj, type) {
	if(typeof obj === null || typeof obj !== 'object' && typeof obj !== 'function') return false;
	let proto = Object.getPrototypeOf(obj);
	let prototype = type.prototype;
	while(proto) {
		if (proto === prototype) return true;
		proto = Object.getPrototypeOf(proto);
	}
	return false;
}
```

```js
		const result = [];
```

### 手写assign
Object的静态方法 Object.assign()
```js
function myAssign(target, ...sources) {
	if(target === null) {
		throw new TypeError('Cannot convert undefined or null to object');
	}
	const res = Object(target);
	sources.forEach(obj => {
		if (obj !== null) {
			for(let key in obj) {
				if (obj.hasOwnProperty(key)) {
					res[key] = obj[key]
				}
			}
		}
	})
	return res;
}
```

```js
Array.prototype.myFilter = function(callback, thisArg) {
	const arr = Array.prototype.slice.call(this);
	let res = [];
	for (let i = 0 ;i < arr.length; i++) {
```

### 手写every
Array的实例方法 Array.prototype.every()

- every(callback(element,index,array),thisArg): Boolean
-  如果回调函数的每一次返回都为 truthy 值，返回 true ，否则返回 false
- 注意：若收到一个空数组，此方法在一切情况下都会返回 true。
- every 不会改变原数组。

```js
Array.prototype.myEvery = function (fn, thisArg) {
	const arr = this;
	for(let i = 0;i < arr.length;i++) {
		if(!fn.call(thisArg,arr[i],i,arr)) {
			return false;
		}
	}
	return true;
}
```

### 题外话 —— Array实例方法对比 & callback参数协议
| 方法      | 本质              |
| ------- | --------------- |
| forEach | 遍历 + 执行         |
| map     | 遍历 + return 新数组 |
| filter  | 遍历 + 条件筛选       |
| reduce  | 遍历 + 累加器        |
| some    | 遍历 + 短路 true    |
| every   | 遍历 + 短路 false   |

参数协议：

reduce、map、filter、forEach都有固定的参数协议
#### map

```
callback(value, index, array)
```

#### filter

```
callback(value, index, array)
```

#### reduce

```
callback(prev, cur, index, array)
```

#### forEach

```
callback(currentValue, index, array)
```
